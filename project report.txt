1. INTRODUCTION
An ATM (Automated Teller Machine) is an electronic device that allows bank customers to perform financial transactions like withdrawal, deposit, balance checking, etc., without visiting the bank.
This project is a simple console-based ATM simulator created using Python. It demonstrates the use of basic programming concepts such as variables, input/output, conditional statements (if-elif-else), and loops (while).

2. OBJECTIVE

To create a menu-driven program in Python
To implement real-life banking operations
To practice the use of loops and decision-making statements
To understand user input handling and basic validation
